# template_parcel
バンドラー parcelのテンプレート  
随時更新予定

# npm script
```npm run dev ```
開発モード  
parcelが起動してwebサーバーが立ち上がる

```npm run build ```

productionモード  
distディレクトリを一度cleanにしてからソースをビルドする。  
ソースマップを作らない。

```npm run clean ```
distディレクトリをcleanにする
